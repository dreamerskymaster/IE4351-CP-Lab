# IE 4350 - CP Manufacturing Lab

This repository contains materials for the **IE 4350: Cyber-Physical Manufacturing Lab** at **Northeastern University**.

## Overview
This project provides an interactive presentation designed for easier understanding of the CP Manufacturing Lab concepts, including:
- Industry 4.0 and Smart Manufacturing principles.
- Cyber-Physical Systems (CPS).
- Modular Architecture and Networked Production.
- Manufacturing Execution Systems (MES).
- Robotic Assembly and AS/RS Systems.

## Presentation
The main presentation can be found in `lab1.html`. It covers:
- Laboratory Safety Requirements.
- Production Workflow Examples.
- Detailed breakdown of Base and Application Modules.
- Integration of Manufacturing Equipment (CNC, 3D Printing, Laser).
